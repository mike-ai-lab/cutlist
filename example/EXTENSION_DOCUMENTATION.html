AutoNestCut — User Documentation

Quick reference and full feature summary. Uses the same visual branding as the extension UI.

Overview

AutoNestCut analyzes your SketchUp model and automatically generates optimized nesting diagrams and detailed cut lists for sheet goods (plywood, MDF, etc.). This document covers features, workflow, configuration tips, and troubleshooting — with screenshots from the example/ folder.

Main features

  • Automatic part extraction (nested components supported)
  • Material database and pricing
  • Optimized nesting with optional rotation and kerf handling
  • Interactive visual diagrams and 3D previews
  • Exports: PDF, CSV, Interactive HTML

Quick actions

  • Load wrapper during development: extension/load_extension.rb
  • Reload extension quickly: extension/reload_extension.rb
  • Production packaging: keep folder structure intact and include AutoNestCut/AutoNestCut/resources/icon.png

Detailed Feature Breakdown

Automatic Part Extraction

AutoNestCut scans selected entities (or the whole model) and extracts parts meeting sheet-good criteria (thickness, area, aspect ratio). Nested components are recursively searched.

Material Management

Maintain a database of stock materials (dimensions and price). The extension detects materials via component attributes, face materials, or component names. Configure custom material sizes and pricing in the settings dialog.

Nesting Algorithm

Uses a greedy bin-packing approach: sort parts by area, attempt placements, allow rotation if permitted, and create new boards when needed. Kerf width and grain restrictions are respected.

Visual Diagrams & 3D Preview

Generates labeled cutting diagrams and an interactive HTML report with 3D part previews. Click parts in diagrams to inspect in 3D; diagrams show board numbering, part labels, and efficiency metrics.

Exports

  • PDF — print-ready diagrams and reports
  • CSV — part lists and inventory data
  • Interactive HTML — shareable reports with embedded 3D views

Troubleshooting & Tips

  • If no parts are found: ensure components have realistic thickness and sufficient area.
  • For better nesting results: allow rotation where grain direction permits and set correct kerf width.
  • Use the material highlight controls in the UI to verify assignments before nesting.
