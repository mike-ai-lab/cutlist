# Reports & Exports

AutoNestCut produces both interactive and static reports suitable for workshop use and sharing.

## Interactive HTML Report

- Includes per-board diagrams, part labels, and an interactive 3D preview for selected parts.
- Ideal for sharing with clients or archiving project output.
- Example report included in the repo: `example/AutoNestCut_Interactive_Report.html`.

## PDF Export

- Creates print-ready diagrams and tables for workshop walls or binders.

## CSV Export

- Exports part lists and board summaries for spreadsheets and inventory systems.

## Best practices

- Use the interactive HTML report for visual verification and quick client reviews.
- Use CSV to import part lists into your shop management spreadsheets.
