# Contributing

Thanks for considering contributing to AutoNestCut. This page explains how you can help.

## Reporting issues

- Open an issue on the repository with a concise description, steps to reproduce, and example model files when possible.

## Code contributions

- Fork the repository, create a feature branch, and open a pull request. Keep changes focused and include tests where feasible.

## Documentation

- Improve or translate the docs in `example/docs/` and open a PR.

## Media and examples

- If you add images for the docs, upload them to the media repo and reference them using the `{{IMAGE_BASE}}` placeholder, and leave local copies in `example/` as a fallback.

Thank you for supporting the project!
